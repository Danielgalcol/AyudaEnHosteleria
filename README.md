# AyudaEnHosteleria

Es una app que ayuda al sector hostelero en su rapidez y coordinación para una mejora en la atención de la clientela.

## Tecnologías utilizadas

- Java
- Android Studio
- Firebase
- Firestore

## Requisitos

- Android Studio
- JDK
- SDK Android
- Cuenta Firebase

## Instalación:

### 1. Clonar repositorio

```bash
git clone "https://github.com/Danielgalcol/AyudaEnHosteleria" 
```
# Compilación del proyecto

## Requisitos previos

- Android Studio instalado
- SDK de Android configurado
- Conexión a Internet
- Cuenta Firebase configurada

## Pasos para compilar

1. Descargar o clonar el proyecto con el comando previamente dicho. Y si se quiere descargar se debeía pulsar en el boton "Code" y "Download.ZIP", lo extraes y ahora en Android Studio le das a "Open" y eligues ese proyecto. 

2. Esperar a que Gradle sincronice las dependencias.

3. Para ejecutar la aplicación:
Run > Run 'app'

4. Para generar el APK:
Build > Generate APKs or App Bundles

## Ubicación del APK generado

En un popup que sale a la derecha abajo te pone "locate" en azul le das ahí y te lleva directamente al APK.
