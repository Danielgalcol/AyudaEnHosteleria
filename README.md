# AyudaEnHosteleria

Es una app que ayuda al sector hostelero en su rapidez y coordinación para una mejora en la atención de la clientela.

## Tecnologías utilizadas

- Java
- Android Studio
- Firebase
- Firestore

## Requisitos

- Android Studio
- JDK
- SDK Android
- Cuenta Firebase
